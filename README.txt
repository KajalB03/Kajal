## 📂 Projects

### 🔹 Economic Shifts in Calgary  
*Tools: Python, Pandas, Seaborn*

### 🔹 Statistical Inference in a Cocaine Drug Bust  
*Tools: R, Probability Theory*

*Tools: R, Regression Modeling*

*Tools: SQL, Data Visualization*
